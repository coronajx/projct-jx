Ando perdido,
no tengo polo sur.
Se han confundido
las marcas de mi juventud.
Y ahí flotando en la ansiedad
se van volando, van…
Córtame, toda la piel.
Quiero realmente estar desnudo
Otra vez me alejé…

que está nublado de raíz.
